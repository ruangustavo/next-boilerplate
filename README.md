# Next.js boilerplate

Esse projeto foi criado com o intuito de facilitar a criação de novos projetos utilizando Next.js.

## Instalação

1. Execute o comando:
  `pnpm create next-app -e https://github.com/ruangustavo/next-boilerplate`

## Tecnologias

- Next.js
- Typescript
- Shadcn/UI
- TailwindCSS
- ESLint
